# Tutorial 2: Transfer Assets Between Wallets

**Network:** Miden Testnet  
**Cost:** $0  
**Prerequisite:** Complete Tutorial 1

## Steps

1. Go to [Miden Playground](https://playground.miden.xyz)
2. Connect your Miden wallet
3. Click **Start Tutorial** for "Transfer assets between wallets"
4. Scroll down and click **Next Step**

### Create Transaction
- Go to **Accounts** (left sidebar)
- Select **Wallet A**
- Click **Create New Transaction** → **New Send Transaction**
- Choose **Wallet B** as recipient
- Select **MDN** asset
- Enter any amount
- Click **Preview** → **Submit**
- Wait for success

### Consume Note
- Go to **Notes**
- Click the first note address
- Select **Consume Note with Wallet B**
- Click **Submit**
- Wait for success
